# 향상된 for문
`조건 : JDK1.5 이상`

지금까지 기본 for문만 사용하여 향상된 for문은 분석밖에 할 수 없을 것 같아 게시글을 작성하게 되었다.

<br>


> for(타입 변수명: 배열 또는 컬렉션) { <br>
>    // 배열 또는 컬렉션에 저장된 값이 매 반복마다 하나씩 순서대로 읽혀져 변수에 저장 <br>
> }

**예시**

```java
for(int k:score) {
    System.out.println(k+" ");
}
```

score의 리스트를 순서대로 k 변수에 대입하여 반복문 진행
